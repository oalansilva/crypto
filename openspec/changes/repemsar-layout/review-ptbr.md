# Revisão - Repemsar Layout

## Resumo

Mudança para padronizar a identidade visual do dashboard crypto e garantir responsividade em todas as páginas.

## O que será feito

1. **Sistema de Design**: Criar variáveis CSS para cores, tipografia, espaçamento e bordas
2. **Navegação Unificada**: Um único componente de menu que funciona em todas as páginas (hamburger no mobile, barra no desktop)
3. **Layout Responsivo**: Todas as 8 páginas adaptam para mobile/tablet/desktop
4. **Cleanup de Headers**: Remover headers customizados de páginas que não usam o AppNav

## Páginas afetadas

- / (home)
- /favorites
- /monitor
- /kanban
- /lab
- /arbitrage
- /combo/select
- /external/balances

## Escopo

- Apenas mudanças visuais/de estilo (sem mudanças de funcionalidade)
- Não adiciona novas páginas
- Não modifica APIs do backend
