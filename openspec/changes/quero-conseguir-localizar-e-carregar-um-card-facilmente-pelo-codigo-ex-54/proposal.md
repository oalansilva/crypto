# Proposal: Localizar Card pelo Código

**Card:** #63  
**change_id:** `find-card-by-code`

---

## Resumo

Permitir que o usuário localize e carregue um card rapidamente digitando seu código (ex: #54). Funcionalidade de busca/jump para cards do Kanban.

## User Story

> Como usuário do Kanban, quero digitar #54 e ir direto para o Card #54 para facilitar a navegação entre cards.

---

## Escopo

### Dentro do escopo
- [ ] Input de busca no Kanban (campo de texto ou atalho)
- [ ] Detectar padrão #XX (hashtag + número)
- [ ] Localizar e abrir/focar o card correspondente
- [ ] Se não encontrado, mostrar mensagem

### Fora do escopo
- [ ] Busca por texto/descrição (apenas por código)
- [ ] Filtros avançados
