# Proposal Review — separar o sistema kanban do projeto crypto

## Resumo

**Problema:** Kanban integrado ao projeto crypto, precisa suportar múltiplos projetos.

**Solução:**
- Seed do projeto crypto já existe no DB
- API para listar/criar projetos
- Selector no frontend para escolher projeto
- Filtragem de cards por projeto

**Escopo MVP:**
- Seed crypto + selector + API básica
- Fora: permissões, mover cards entre projetos

**Próximo passo:** Aprovação → DEV → QA → Homologação.
