## Dados Minimos Antes do Convite

Para cada pessoa, registrar antes de convidar:

- canal preferencial: WhatsApp, Telegram, LinkedIn, Instagram ou outro;
- se ja investe ou acompanha cripto;
- nivel aproximado: iniciante, intermediario ou trader;
- principal motivo para convidar;
- se Alan quer convite direto ou conversa previa;
- status do convite.

## Status Possiveis

- `Sugerido`: nome indicado por Alan.
- `Contato definido`: canal de contato registrado.
- `Convite pronto`: texto aprovado para essa pessoa/canal.
- `Convite enviado`: Alan enviou manualmente.
- `Respondeu`: pessoa respondeu.
- `Entrou na lista`: pessoa entrou pelo site.
- `Tester ativo`: pessoa acessou/testou o beta.
- `Pausado`: nao acionar agora.
